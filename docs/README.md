# MR Bus Portal Ultimate

A full-stack intelligent bus booking platform that allows users to search routes, book tickets, and interact with an AI-powered system for travel planning.

## Tech Stack
Backend: FastAPI (Python)
Frontend: React.js
AI Integration: OpenAI API
Database: PostgreSQL
Deployment: Docker / Cloud Ready

## Features
- OTP-based authentication
- Nationwide bus dataset generator
- Bus search API
- Ticket booking system
- Modular microservice ready architecture
